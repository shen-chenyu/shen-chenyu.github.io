### [2D Transport in an in-plane magnetic field](https://arxiv.org/abs/2607.06561v1)
**Authors:** Aryaman Babbar, Sankar Das Sarma
**Published:** 2026-07-07
**Summary:** In-plane magnetic field distinguishes Wigner crystallization from Anderson localization in 2D metal-insulator transitions via spin polarization effects on screening.

**What's new**
- Theory shows in-plane B-field enhances critical density for Anderson localization by reducing screening via spin polarization.
- Critical density may decrease if short-ranged defects dominate scattering, unlike long-ranged charged impurities.
- Spin polarization affects screening (enhances density) vs. Fermi surface (suppresses density) differently.
- Quantitative predictions match two recent experiments on 2D MIT.

### [Graph Convolutional Attention: A Spectral Perspective on Graph Denoising and Diffusion](https://arxiv.org/abs/2607.06546v1)
**Authors:** Shervin Khalafi, Igor Krawczuk, Sergio Rozada, Charilaos Kanatsoulis, Antonio G Marques, Alejandro Ribeiro
**Published:** 2026-07-07
**Summary:** Introduces Graph Convolutional Attention (GCA), a spectral-aware attention mechanism that provably outperforms linear attention for graph denoising and diffusion.

**What's new**
- Shows linear attention under denoising objective learns only an average spectral filter, limiting performance on spectrally diverse graphs.
- Proposes Spectral Attention that uses input graph spectrum to overcome this limitation.
- Derives GCA, a permutation-equivariant realization using graph-filtered queries and keys.
- Proves GCA matches idealized Spectral Attention for stochastic block models.
- Empirically demonstrates GCA improves graph denoising and diffusion, matching transformer performance without expensive structural features.

### [DepthWeave-KV: Token-Adaptive Cross-Layer Residual Factorization for Long-Context KV Cache Compression](https://arxiv.org/abs/2607.06523v1)
**Authors:** Anna Cordoba, Adam Puente Tercero, Nerea Angulo Hijo, Mar Linares Tercero, Julia Barrientos, Ainhoa Miranda, Jesus Olivera
**Published:** 2026-07-07
**Summary:** DepthWeave-KV compresses long-context KV caches via token-adaptive cross-layer residual factorization, achieving 8.3x memory reduction with near-full-cache quality.

**What's new**
- Token-adaptive compression: allocates higher rank to instruction-bearing and retrieval-critical tokens.
- Cross-depth residual factorization: shares low-rank bases across layers, with token-specific residuals for sensitive attention.
- Calibration-free online error tracking: uses attention-output probes to adapt compression during generation.
- Fused CUDA kernel: jointly performs basis lookup, residual dequantization, and attention projection to reduce memory traffic.
- 8.3x KV memory reduction and 72.8 tokens/sec at 64K context, with improved retrieval accuracy over prior methods.
