### [Cortex: A Bidirectionally Aligned Embodied Agent Framework for Long-horizon Manipulation](https://arxiv.org/abs/2607.05377v1)
**Authors:** Jiaqi Peng, Xiqian Yu, Delin Feng, Yuqiang Yang, Wenzhe Cai, Jing Xiong, Ganlin Yang, Jinliang Zheng, Jiafei Cao, Xueyuan Wei, Jiangmiao Pang, Yuan Shen, Tai Wang
**Published:** 2026-07-06
**Summary:** Cortex is a bidirectionally aligned embodied agent framework that bridges high-level planning and low-level execution for long-horizon manipulation tasks.

**What's new**
- Standardizes manipulation subtasks into 32 canonical skill primitives with tractability principles.
- Automatically annotates over 4k hours of open-source video and generates 30 hours of simulation data.
- Uses event-balanced sampling to fine-tune the framework for handling planning ambiguity during subtask transitions.
- Outperforms monolithic baselines by 3.1% on Libero-long and 4.1% on RoboTwin.
- Enables zero-shot completion of unseen real-world long-horizon tasks by combining a generalist VLM with a fine-tuned VLA.

### [LLM-as-a-Verifier: A General-Purpose Verification Framework](https://arxiv.org/abs/2607.05391v1)
**Authors:** Jacky Kwok, Shulu Li, Pranav Atreya, Yuejiang Liu, Yixing Jiang, Chelsea Finn, Marco Pavone, Ion Stoica, Azalia Mirhoseini
**Published:** 2026-07-06
**Summary:** LLM-as-a-Verifier is a general-purpose verification framework that uses probabilistic scoring to scale verification accuracy across agentic tasks.

**What's new**
- Introduces a probabilistic formulation using scoring token logits for continuous, fine-grained verification scores.
- Demonstrates scaling verification along score granularity, repeated evaluation, and criteria decomposition for improved accuracy.
- Achieves state-of-the-art results on Terminal-Bench V2, SWE-Bench Verified, RoboRewardBench, and MedAgentBench.
- Provides a cost-efficient ranking algorithm for selecting the best solution among candidates using continuous scores.
- Shows that fine-grained verification signals can serve as a proxy for task progress and improve RL sample efficiency.

### [MetaSkill-Evolve: Recursive Self-Improvement of LLM Agents via Two-Timescale Meta-Skill Evolution](https://arxiv.org/abs/2607.05297v1)
**Authors:** Zefeng Wang, Minxi Yan, Jinhe Bi, Sikuan Yan, Volker Tresp, Yunpu Ma
**Published:** 2026-07-06
**Summary:** MetaSkill-Evolve enables LLM agents to recursively improve both task skills and the meta-skill that governs improvement, boosting performance on benchmarks.

### [Fitted Occupancy-Ratio Evaluation without Bellman Completeness](https://arxiv.org/abs/2607.05375v1)
**Authors:** Lars van der Laan, Nathan Kallus
**Published:** 2026-07-06
**Summary:** FORE estimates occupancy ratios via a fitted fixed-point method, requiring only ratio realizability, not Bellman completeness.

**What's new**
- Introduces FORE, a fitted fixed-point method for occupancy-ratio evaluation using adjoint Bellman recursion.
- Requires only realizability of the discounted occupancy ratio, avoiding Bellman completeness assumptions.
- Proves KL contraction of the population recursion and finite-sample regret bounds for the empirical recursion.
- Supports direct value estimation, occupancy-weighted FQE, and doubly robust estimation with fitted ratio and Q-function.
